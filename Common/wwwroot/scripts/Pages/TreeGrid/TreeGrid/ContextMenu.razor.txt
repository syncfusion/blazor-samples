<ActionDescription>
<p>
    This sample demonstrates the Tree Grid context menu, which provides contextual actions such as editing, deleting, exporting, sorting, copying, paging, and column sizing through a right-click menu on rows, headers, and the pager.
  </p>
  <p>
    Enable the context menu by setting <a target="_blank" rel="noopener" aria-label="Navigate to the ContextMenuItems property reference for the Tree Grid component" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.TreeGrid.SfTreeGrid-1.html#Syncfusion_Blazor_TreeGrid_SfTreeGrid_1_ContextMenuItems">ContextMenuItems</a> to a collection of default or custom items. Items render based on the clicked target (header, row, or pager).
  </p>
  <p>
    In this demo, a self-referential support ticket dataset is bound. The context menu contains:
  </p>
  <p><strong>Content (Row):</strong></p>
  <ul>
    <li>Edit</li>
    <li>Delete</li>
    <li>Save / Cancel (visible during edit)</li>
    <li>Copy</li>
    <li>Export (PDF, Excel, CSV)</li>
  </ul>
  <p><strong>Header:</strong></p>
  <ul>
    <li>Sort Ascending / Sort Descending</li>
    <li>AutoFit</li>
    <li>AutoFitAll</li>
  </ul>
  <p><strong>Pager:</strong></p>
  <ul>
    <li>First Page, Previous Page, Next Page, Last Page</li>
  </ul>
  <p>
    Learn more in the <a target="_blank" rel="noopener" aria-label="Navigate to the documentation for context menu in the Tree Grid component" href="https://blazor.syncfusion.com/documentation/treegrid/context-menu">context menu documentation</a>.
  </p>
</ActionDescription>
