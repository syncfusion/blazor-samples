<ActionDescription>
<p>
    This sample demonstrates the toolbar shown in inline mode. The toolbar appears when you select the editable content and hides when focus is removed from the edit area.
  </p>
  <p>
    The <a href="https://www.syncfusion.com/blazor-components/blazor-wysiwyg-rich-text-editor" aria-label="Feature Tour of Blazor Rich Text Editor">Blazor Rich Text Editor</a> provides an option to display the toolbar on demand by setting the <code>Enable</code> property of <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.RichTextEditor.RichTextEditorInlineMode.html" aria-label="Class reference of RichTextEditorInlineMode">RichTextEditorInlineMode</a>. The commands displayed in the inline toolbar can be customized by setting the <code>Items</code> property of <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.RichTextEditor.RichTextEditorToolbarSettings.html" aria-label="Class reference of RichTextEditorToolbarSettings">RichTextEditorToolbarSettings</a>.
  </p>
  <p>
    <strong>Supported Keyboard Shortcuts:</strong>
  </p>
  <p><strong>FORMATTING</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>B</kbd></span> - Applies bold formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>I</kbd></span> - Applies italic formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>U</kbd></span> - Applies underline formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd></span> - Applies strikethrough formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>]</kbd></span> - Increases the indent.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>[</kbd></span> - Decreases the indent.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd></span> - Changes text to uppercase.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd></span> - Changes text to lowercase.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>=</kbd></span> - Applies superscript formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>=</kbd></span> - Applies subscript formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd></span> - Applies ordered list formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>O</kbd></span> - Applies unordered list formatting.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd></span> - Clears formatting from the selection.</li>
  </ul>
  <p><strong>ALIGNMENTS</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>E</kbd></span> - Aligns content to the center.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>J</kbd></span> - Aligns content to justify.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>L</kbd></span> - Aligns content to the left.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>R</kbd></span> - Aligns content to the right.</li>
  </ul>
  <p><strong>CLIPBOARD ACTION</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>C</kbd></span> - Copies selected content from the editor.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>X</kbd></span> - Cuts selected content from the editor.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>V</kbd></span> - Pastes content into the editor.</li>
  </ul>
  <p><strong>INSERT</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>K</kbd></span> - Opens the insert link dialog.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd></span> - Opens the insert image dialog.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd></span> - Opens the insert table dialog.</li>
  </ul>
  <p><strong>UNDO/REDO</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Z</kbd></span> - Performs the undo action.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Y</kbd></span> - Performs the redo action.</li>
  </ul>
  <p><strong>FULLSCREEN</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd></span> - Enters full-screen mode.</li>
    <li><span class="key-class"><kbd>Esc</kbd></span> - Exits full-screen mode.</li>
  </ul>
  <p><strong>SOURCE CODE VIEW</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd></span> - Shows the HTML source.</li>
  </ul>
</ActionDescription>
