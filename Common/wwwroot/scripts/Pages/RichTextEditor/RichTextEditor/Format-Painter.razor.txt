<ActionDescription>
<p>
    This demo showcases the Format Painter feature of the Rich Text Editor component, which allows users to copy formatting from one section of content and apply it to another.
  </p>
  <p>
    The <strong>Format Painter</strong> feature allows you to copy formats and apply them to unformatted content, saving time on manual reformatting:
  </p>
  <ul>
    <li>Format painter can be accessed via the toolbar or keyboard shortcuts.</li>
    <li>Sticky mode can be enabled by double-clicking the toolbar button to apply formatting to multiple locations.</li>
  </ul>
  <p>
    <strong>Keyboard Shortcuts:</strong>
  </p>
  <ul>
    <li><span class="key-class"><kbd>ALT</kbd> + <kbd>SHIFT</kbd> + <kbd>C</kbd></span> - Copy the selection format on current range.</li>
    <li><span class="key-class"><kbd>ALT</kbd> + <kbd>SHIFT</kbd> + <kbd>V</kbd></span> - Paint the copied format.</li>
    <li><span class="key-class"><kbd>ESC</kbd></span> - Remove the previously copied format and disable sticky mode.</li>
  </ul>
  <p>
    The following settings are available to customize the format painter in the <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.RichTextEditor.RichTextEditorFormatPainterSettings.html" aria-label="Class reference of RichTextEditorFormatPainterSettings">RichTextEditorFormatPainterSettings</a> property:
  </p>
  <ul>
    <li>Fill the <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.RichTextEditor.RichTextEditorFormatPainterSettings.html#Syncfusion_Blazor_RichTextEditor_RichTextEditorFormatPainterSettings_AllowedFormats" aria-label="Class reference of Allowed Formats property in Rich Text Editor">Allowed Formats</a> input with selectors only whose format styles will be allowed. For example: <code>span; strong; em;</code> allows only span, strong, and em format styles to be copied.</li>
    <li>Fill the <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.RichTextEditor.RichTextEditorFormatPainterSettings.html#Syncfusion_Blazor_RichTextEditor_RichTextEditorFormatPainterSettings_DeniedFormats" aria-label="Class reference of Allowed Formats property in Rich Text Editor">Denied Formats</a> input with selectors only whose format styles will be explicitly prohibited. For example: <code>span(important)[title]{background-color,color}</code> will remove only the <code>important</code> class, <code>title</code> attribute, and <code>color</code> and <code>background-color</code> properties of the span element.</li>
  </ul>
</ActionDescription>

@code {
    private string? allowedFormatText { get; set; }
    private string? allowedFormats { get; set; } = "b; em; font; sub; sup; kbd; i; s; u; code; strong; span; p; div; h1; h2; h3; h4; h5; h6; blockquote; ol; ul; li; pre;";
    private string? deniedFormats { get; set; } = null;
    private string? deniedFormatText { get; set; }

    private void allowedFormatsPainterBlur ( Microsoft.AspNetCore.Components.Web.FocusEventArgs e )
    {
        if (!string.IsNullOrWhiteSpace (this.allowedFormatText))
        {
            this.allowedFormats = this.allowedFormatText;
        }
        else
        {
            this.allowedFormats = "b; em; font; sub; sup; kbd; i; s; u; code; strong; span; p; div; h1; h2; h3; h4; h5; h6; blockquote; table; thead; tbody; tr; td; th; ol; ul; li; pre; (*)[*]{*}";
        }
    }

    private void deniedFormatPainterBlur ( Microsoft.AspNetCore.Components.Web.FocusEventArgs e )
    {
        if (!string.IsNullOrWhiteSpace (this.deniedFormatText))
        {
            this.deniedFormats = this.deniedFormatText;
        }
        else
        {
            this.deniedFormats = null;
        }
    }
