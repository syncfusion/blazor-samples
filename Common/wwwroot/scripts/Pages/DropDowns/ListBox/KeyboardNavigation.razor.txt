<SampleDescription>
    <p>This sample demonstrates the keyboard interactions of the ListBox component. The various key combinations can be
        used to interact with ListBox.</p>
</SampleDescription>
<ActionDescription>
    <i>The ListBox supports the following keyboard shortcuts to initiate various actions.</i>
    <ul>
        <li>
            <b>Supported keys and user actions</b>
            <ul>
                <li>
                    <span class="key-class"><kbd>Alt</kbd> + <kbd>J</kbd></span>
                    <span> - Focuses on the first component of the demo.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Up arrow</kbd></span>
                    <span> - Moves focus to the previous option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Down arrow</kbd></span>
                    <span> - Moves focus to the next option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Home</kbd></span>
                    <span> - Moves focus to first option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>End</kbd></span>
                    <span> - Moves focus to last option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Space</kbd></span>
                    <span> - Changes the selection state of the focused option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Ctrl</kbd> + <kbd>A</kbd></span>
                    <span> - Selects all options in the list.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Home</kbd></span>
                    <span> - Selects the focused option and all options up to the first option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>End</kbd></span>
                    <span> - Selects the focused option and all options down to the last option.</span>
                </li>
                <li>
                    <span class="key-class"><kbd>Ctrl</kbd> + <kbd>Arrow keys</kbd></span>
                    <span> - Press Ctrl key with up / down arrow or mouse to select multiple items.</span>
                </li>
            </ul>
        </li>
    </ul>
</ActionDescription>
