<ActionDescription>
<p>
    The Blazor ListView component supports rich keyboard interactions. Use the following key combinations to navigate and interact with the list.
  </p>
  <ul>
    <li>
      <strong>Focus</strong>
      <ul>
        <li><span class="key-class"><kbd>Alt</kbd> + <kbd>J</kbd></span> - Focuses the first component of the sample.</li>
      </ul>
    </li>
    <li>
      <strong>Supported keys and actions</strong>
      <ul>
        <li><span class="key-class"><kbd>Up arrow</kbd> or <kbd>Left arrow</kbd></span> - Moves to the previous list item.</li>
        <li><span class="key-class"><kbd>Down arrow</kbd> or <kbd>Right arrow</kbd></span> - Moves to the next list item.</li>
        <li><span class="key-class"><kbd>Space</kbd> or <kbd>Enter</kbd></span> - Selects the targeted list item when <a target="_blank" rel="noopener" class="code" href="https://help.syncfusion.com/cr/cref_files/blazor/Syncfusion.Blazor~Syncfusion.Blazor.Lists.SfListView%601~ShowCheckBox.html" aria-label="Class reference of ShowCheckBox property in ListView">ShowCheckBox</a> is enabled.</li>
        <li><span class="key-class"><kbd>Back</kbd></span> - Navigates back to the previous list (nested lists).</li>
        <li><span class="key-class"><kbd>Home</kbd></span> - Moves to the first item.</li>
        <li><span class="key-class"><kbd>End</kbd></span> - Moves to the last item.</li>
      </ul>
    </li>
  </ul>
</ActionDescription>
