<ActionDescription>
<p>
    This sample demonstrates binding remote data to the ListView component. Click any item to select and highlight it.
  </p>
  <p>
    The <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Data.SfDataManager.html" aria-label="Class reference of DataManager">DataManager</a> acts as an interface between the service endpoint and ListView, and requires:
  </p>
  <p>
    <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DataManager.html#Syncfusion_Blazor_DataManager_Url" target="_blank" rel="noopener" aria-label="Class reference of Url property in DataManager">DataManager.Url</a> – Endpoint to fetch data.
  </p>
  <p>
    <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DataManager.html#Syncfusion_Blazor_DataManager_Adaptor" target="_blank" rel="noopener" aria-label="Class reference of Adaptor property in DataManager">DataManager.Adaptor</a> – Adaptor type (default: ODataAdaptor) which processes requests and responses.
  </p>
  <p>
    The <code>Syncfusion.Blazor.Data</code> namespace provides predefined adaptors for various endpoints:
  </p>
  <ul>
    <li><strong>UrlAdaptor</strong> – Generic remote services</li>
    <li><strong>ODataAdaptor</strong> – OData endpoints</li>
    <li><strong>ODataV4Adaptor</strong> – OData V4 endpoints</li>
    <li><strong>WebApiAdaptor</strong> – Web API (OData standards)</li>
    <li><strong>WebMethodAdaptor</strong> – Web methods</li>
  </ul>
  <p>
    In this sample, remote <code>Products</code> data is bound via a <code>DataManager</code> instance. Learn more in the <a target="_blank" rel="noopener" class="code" href="https://blazor.syncfusion.com/documentation/listview/data-binding#bind-to-remote-data" aria-label="Blazor ListView DataBinding documentation">documentation</a>.
  </p>
</ActionDescription>


<div class="col-lg-12 control-section">
    @* Initialization of ListView component with remote data *@
    <SfListView CssClass="listview" HeaderTitle="Products" ShowHeader="true" TValue="DataModel" Query="@query">
        <ListViewFieldSettings TValue="DataModel" Id="ProductID" Text="ProductName"></ListViewFieldSettings>
        <SfDataManager Url="https://services.odata.org/V4/Northwind/Northwind.svc/" Adaptor="Adaptors.ODataV4Adaptor"></SfDataManager>
    </SfListView>
</div>

@code{
    // Specifies the query string to fetch the datasource.
    Query query = new Query().From("Products").Select(new List<string>() { "ProductID","ProductName"}).Take(10);
    
    // Specifies the model class for ListView datasource.
    public class DataModel
    {
        public int ProductID { get; set; }
        public string? ProductName { get; set; }
    }

}
