<ActionDescription>
<p>This sample demonstrates the DataGrid automatic cell merging feature for combining adjacent cells with identical values.</p>
  
  <p>The Blazor DataGrid supports automatic cell merging, allowing adjacent cells with identical values to be combined horizontally, vertically, or in both directions. This creates a cleaner and more readable data layout.</p>
  
  <p>The cell merging behavior is controlled by the <strong>AutoSpan</strong> property, which is bound to a dropdown selector in this demo. Available merging modes are:</p>
  <ul>
    <li><strong>None</strong> - Disables cell merging; displays original grid layout</li>
    <li><strong>Row</strong> - Merges identical cells horizontally across columns within the same row</li>
    <li><strong>Column</strong> - Merges identical cells vertically down the same column</li>
    <li><strong>HorizontalAndVertical</strong> - Applies both horizontal and vertical cell merging simultaneously (default)</li>
  </ul>
  
  <p>Select different modes from the dropdown to see how the cell merging behavior changes. By default, <code>AutoSpanMode.HorizontalAndVertical</code> is applied, providing optimal readability by merging identical cells across both rows and columns.</p>
  
  <p>For more detailed information about row and column spanning, refer to the <a target="_blank" rel="noopener" href="https://blazor.syncfusion.com/documentation/datagrid/row">row</a> and <a target="_blank" rel="noopener" href="https://blazor.syncfusion.com/documentation/datagrid/columns">column</a> spanning documentation.</p>
</ActionDescription>

@code {
    private AutoSpanMode currentMode = AutoSpanMode.HorizontalAndVertical;
    private string selectedMode = "Horizontal and Vertical";
    private List<string> modes = new() { "None", "Row", "Column", "Horizontal and Vertical" };
    private bool enableFrozen { get; set; } = true;
    @*Hidden:Lines*@
    private bool isDeviceMode { get; set; }
    @*End:Hidden*@
    public List<EmployeeDetails>? EmployeeTimeSheet { get; set; }

    protected override void OnInitialized()
    {
        EmployeeTimeSheet = EmployeeDetails.GetAllRecords();
    }

    @*Hidden:Lines*@
    protected override async void OnAfterRender(bool firstRender)
    {
        base.OnAfterRender(firstRender);
        if (firstRender)
        {
            isDeviceMode = await this.JS.InvokeAsync<bool>("sfBlazorSB.isDeviceMode");
            if (isDeviceMode)
            {
                enableFrozen = false;
                StateHasChanged();
            }
        }
    }
    @*End:Hidden*@

    private void OnModeChanged(ChangeEventArgs<string, string> args)
    {
        selectedMode = args?.Value ?? "None";
        currentMode = selectedMode switch
        {
            "Row" => AutoSpanMode.Row,
            "Column" => AutoSpanMode.Column,
            "Horizontal and Vertical" => AutoSpanMode.HorizontalAndVertical,
            _ => AutoSpanMode.None
        };
    }
}
