<ActionDescription>
<p>This sample demonstrates how to bind list objects to the DataGrid component for local data binding.</p>
  
  <p>The Blazor DataGrid supports data binding with local collections. The <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_DataSource' aria-label="Navigate to the DataSource property reference for DataGrid component">DataSource</a> property can be assigned either as a list of objects or as an instance of the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DataManager.html' aria-label="Navigate to the class reference for DataManager">DataManager</a> component.</p>
  
  <p>In this demo, a list of objects is assigned directly as the DataGrid datasource. The DataGrid supports binding to the following data source types:</p>
  <ul>
    <li>IEnumerable - Standard enumerable collections</li>
    <li>IQueryable - LINQ queryable collections</li>
    <li>ObservableCollection - Collections with automatic change notification</li>
  </ul>
  
  <p>For more detailed information about local data binding, refer to the <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/data-binding/#list-binding' aria-label="Navigate to the documentation for list binding in DataGrid component">list binding documentation</a>.</p>
