<ActionDescription>
<p>This sample demonstrates the DataGrid component with auto wrap column cell content functionality.</p>
  
  <p>The <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AllowTextWrap' aria-label="Navigate to the AllowTextWrap property reference for DataGrid component">AllowTextWrap</a> property enables automatic text wrapping in the Blazor DataGrid. When the width of cell content or header text exceeds the available cell width, this property wraps the text into multiple lines, ensuring all content remains visible.</p>
  
  <p>In this demo, the <code>AllowTextWrap</code> property is enabled for the <strong>Main Fields of Invention</strong> column. You can observe how the content is automatically wrapped into multiple lines when it exceeds the cell width.</p>
  
  <p>For more detailed information about the autowrap feature, refer to the <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/cell#autowrap-the-grid-content' aria-label="Navigate to the documentation for Autowrap the grid content">autowrap documentation</a>.</p>
