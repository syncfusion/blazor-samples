<ActionDescription>
<p>This sample demonstrates the DataGrid stacked headers feature with column resize capability for hierarchical header organization.</p>
  
  <p>The Blazor DataGrid supports stacked headers, allowing you to create multiple levels of column headers for logical grouping. Create stacked headers by nesting <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_Columns' aria-label="Navigate to the class reference for GridColumn">GridColumn</a> components within parent <code>GridColumn</code> components. Column resizing is fully supported for both regular and stacked columns.</p>
  
  <p>In this demo:</p>
  <ul>
    <li>The <strong>Order Date</strong> and <strong>Freight</strong> columns are grouped under the <strong>Order Details</strong> header</li>
    <li>The <strong>Shipped Date</strong> and <strong>Ship Country</strong> columns are grouped under the <strong>Shipped Details</strong> header</li>
  </ul>
  
  <p>This hierarchical structure provides better organization and visual clarity for related data columns.</p>
  
  <p>For more detailed information about stacked headers, refer to the <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/column-headers#stacked-header' aria-label="Navigate to the documentation for Stacked header in DataGrid column">stacked header documentation</a>.</p>
</ActionDescription>
