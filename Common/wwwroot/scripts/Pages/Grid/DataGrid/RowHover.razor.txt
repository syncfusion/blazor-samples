<ActionDescription>
<p>This sample demonstrates the DataGrid row hover feature for visual row identification.</p>
  
  <p>The Blazor DataGrid row hover feature highlights the current row when the mouse is hovered over it, improving visual feedback and user experience. To enable this feature, set the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_EnableHover' aria-label="Navigate to the EnableHover property reference for DataGrid component">EnableHover</a> property to <strong>true</strong>.</p>
  
  <p>In this demo, move your mouse over any DataGrid row to see the hover effect applied. The highlighted row makes it easy to identify which row you're currently interacting with, particularly useful in grids with many columns or rows.</p>
  
  <p>This feature is enabled by default in many DataGrid implementations and provides a subtle but effective visual cue for user interactions.</p>
</ActionDescription>
