<SampleDescription>
    <p>This sample demonstrates the frozen rows and columns feature of the DataGrid. Scroll the movable content vertically and horizontally to see the frozen rows and columns respectively with the content.</p>
</SampleDescription>
<ActionDescription>
    <p>The Blazor DataGrid allows you to prevent certain rows and columns from scrolling, leaving only the remaining content scrollable. This is possible by using the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_FrozenRows'>FrozenRows</a> and <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_FrozenColumns'>FrozenColumns</a>  properties.</p>
    <p>In this demo sample, the <b>Title</b> is frozen, and the remaining content can be scrolled horizontally for comparison while the entire row is frozen to compare the content by scrolling vertically.</p>
    <p>More information about the Frozen rows and columns feature can be found in this <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/scrolling/#frozen-rows-and-columns'>documentation</a> section.</p>
</ActionDescription>
