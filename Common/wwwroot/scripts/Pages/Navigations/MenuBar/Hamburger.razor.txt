<ActionDescription>
<p>This sample demonstrates hamburger mode functionality in the Menu component, enabling adaptive menu display.</p>
    
    <p>Hamburger mode features:</p>
    
    <ul>
        <li>Enable adaptive view using the <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Navigations.SfMenu-1.html#Syncfusion_Blazor_Navigations_SfMenu_1_HamburgerMode" aria-label="HamburgerMode">HamburgerMode</a> property</li>
        <li>Displays header with hamburger icon in <b>Horizontal</b> orientation by default</li>
        <li>Menu appears on hamburger icon click</li>
        <li>Programmatic control using <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Navigations.SfMenu-1.html#Syncfusion_Blazor_Navigations_SfMenu_1_OpenAsync" aria-label="OpenAsync">OpenAsync</a> and <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Navigations.SfMenu-1.html#Syncfusion_Blazor_Navigations_SfMenu_1_CloseAsync' aria-label="CloseAsync">CloseAsync</a> methods</li>
    </ul>
    
    <p>For comprehensive hamburger menu documentation, refer to the <a target="_blank" rel="noopener" href="https://blazor.syncfusion.com/documentation/menu-bar/usecase-scenario#hamburger-menu" aria-label="documentation section">documentation</a>.</p>
</ActionDescription>
