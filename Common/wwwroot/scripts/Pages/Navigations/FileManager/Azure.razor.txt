<ActionDescription>
<p>This sample demonstrates integration of the <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/azure-aspcore-file-provider" aria-label="GitHub sample link of Azure file system provider">Azure file system provider</a> with the File Manager component for cloud storage file operations.</p>
    
    <p>Supported file operations include create, rename, delete, cut, copy, paste, upload, download, and more.</p>
    
    <p>Implementation process:</p>
    
    <ul>
        <li>Create an <a target="_blank" rel="noopener" href="https://learn.microsoft.com/en-us/azure/storage/common/storage-account-create?tabs=azure-portal" aria-label="Azure blob storage account documentation">Azure blob storage account</a></li>
        <li>Configure Azure credentials including account name, password, and blob name in the <code>RegisterAzure</code> method</li>
        <li>Use <code>BlobContainerClient</code> class for blob storage access</li>
        <li>Access storage files via <code>GetFilesAsync</code> method for File Manager operations</li>
    </ul>
    
    <p>Refer to the <a target="_blank" rel="noopener" href="https://learn.microsoft.com/en-IN/azure/storage/blobs/storage-quickstart-blobs-dotnet?culture=en-in&country=in&tabs=visual-studio%2Cmanaged-identity%2Croles-azure-portal%2Csign-in-azure-cli%2Cidentity-visual-studio" aria-label="Azure Blob Storage Client Library documentation">Azure Blob Storage Client Library</a> for additional details.</p>
    
    <p>Access the <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/azure-aspcore-file-provider" aria-label="GitHub sample link of Azure file system provider">Azure file system provider repository</a> for implementation resources.</p>
</ActionDescription>
