<ActionDescription>
<p>The <a target="_blank" rel="noopener" href="https://www.syncfusion.com/blazor-components/blazor-file-manager" aria-label="Feature Tour of Blazor File Manager">Blazor File Manager</a> component provides comprehensive file and folder management functionality similar to Windows Explorer.</p>
    
    <p>Core features include:</p>
    
    <ul>
        <li><a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerToolbarSettings.html#properties" aria-label="Class reference of toolbar settings property in File Manager">Toolbar</a> - Essential operation buttons</li>
        <li><a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerNavigationPaneSettings.html#properties" aria-label="Class reference of navigation pane property in File Manager">Navigation Pane</a> - Folder hierarchy display</li>
        <li><a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerDetailsViewSettings.html#properties" aria-label="Class reference of details view property in File Manager">Details View</a> - File information display</li>
    </ul>
    
    <p>Supported file operations: create, rename, delete, cut, copy, paste, upload, download, and refresh.</p>
    
    <p>Configuration requirements:</p>
    
    <ul>
        <li>Define <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerAjaxSettings.html#properties" aria-label="Class reference of ajax settings property in File Manager">AjaxSettings</a> with URLs for file operation requests</li>
        <li>Customize context menu via <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerContextMenuSettings.html#properties" aria-label="Class reference of context menu settings property in File Manager">ContextMenuSettings API</a></li>
        <li>Add custom toolbar items using <a target="_blank" rel="noopener" href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.FileManager.FileManagerToolbarSettings.html#properties" aria-label="Class reference of toolbar settings property in File Manager">ToolbarSettings API</a></li>
    </ul>
</ActionDescription>


<div class="control-section">
    @* Initialization of default File Manager component *@
    <SfFileManager TValue="FileManagerDirectoryContent" AllowDragAndDrop="true" AllowMultiSelection="true" ShowFileExtension="false" >
        <FileManagerAjaxSettings Url="https://ej2-aspcore-service.azurewebsites.net/api/FileManager/FileOperations"
                                 UploadUrl="https://ej2-aspcore-service.azurewebsites.net/api/FileManager/Upload"
                                 DownloadUrl="https://ej2-aspcore-service.azurewebsites.net/api/FileManager/Download"
                                 GetImageUrl="https://ej2-aspcore-service.azurewebsites.net/api/FileManager/GetImage">
        </FileManagerAjaxSettings>
        <FileManagerToolbarSettings ToolbarItems="@Items"></FileManagerToolbarSettings>
        <FileManagerContextMenuSettings File="@FileItems" Layout="@LayoutItems" Visible="true"></FileManagerContextMenuSettings>
        <FileManagerDetailsViewSettings>
            <FileManagerColumns>
                <FileManagerColumn Field="Name" HeaderText="Name"></FileManagerColumn>
                <FileManagerColumn Field="DateModified" Format="MM/dd/yyyy h:mm tt" HeaderText="Modified"></FileManagerColumn>
                <FileManagerColumn Field="Size" HeaderText="Size"></FileManagerColumn>
            </FileManagerColumns>
        </FileManagerDetailsViewSettings>
    </SfFileManager>
</div>
