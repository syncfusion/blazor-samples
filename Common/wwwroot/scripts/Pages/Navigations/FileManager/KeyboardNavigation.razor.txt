<ActionDescription>
<p>This sample demonstrates keyboard accessibility features of the <a target="_blank" rel="noopener" href="https://www.syncfusion.com/blazor-components/blazor-file-manager" aria-label="Feature Tour of Blazor File Manager">Blazor File Manager</a> component through comprehensive keyboard shortcut support.</p>
    
    <p>Keyboard interaction categories and shortcuts:</p>
    
    <ul>
        <li><b>Focus Navigation</b>
            <ul>
                <li><span class="key-class"><kbd>Alt</kbd> + <kbd>J</kbd></span> - Focus the first component</li>
            </ul>
        </li>
        <li><b>Core Operations</b>
            <ul>
                <li><span class="key-class"><kbd>F5</kbd></span> - Refresh component</li>
                <li><span class="key-class"><kbd>F2</kbd></span> - Rename selected item</li>
                <li><span class="key-class"><kbd>Space</kbd></span> - Select focused item</li>
                <li><span class="key-class"><kbd>Home</kbd></span> - Select first item</li>
                <li><span class="key-class"><kbd>End</kbd></span> - Select last item</li>
                <li><span class="key-class"><kbd>Enter</kbd></span> - Open selected item</li>
                <li><span class="key-class"><kbd>Del</kbd> or <kbd>Shift</kbd> + <kbd>Del</kbd></span> - Delete selected item</li>
            </ul>
        </li>
        <li><b>File Operations</b>
            <ul>
                <li><span class="key-class"><kbd>Alt</kbd> + <kbd>N</kbd></span> - Create new folder</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>A</kbd></span> - Select all items</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>D</kbd></span> - Download selected item</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>X</kbd></span> - Cut selected item</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>C</kbd></span> - Copy selected item</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>V</kbd></span> - Paste copied items</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>U</kbd></span> - Open file upload window</li>
                <li><span class="key-class"><kbd>Alt</kbd> + <kbd>Enter</kbd></span> - Display selected item details</li>
            </ul>
        </li>
        <li><b>Navigation and Selection</b>
            <ul>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Home</kbd></span> - Focus first item</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>End</kbd></span> - Focus last item</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>1</kbd></span> - Switch to Details view</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>2</kbd></span> - Switch to LargeIcons view</li>
                <li><span class="key-class"><kbd>Shift</kbd> + <kbd>Home</kbd></span> - Select from current to first item</li>
                <li><span class="key-class"><kbd>Shift</kbd> + <kbd>End</kbd></span> - Select from current to last item</li>
                <li><span class="key-class"><kbd>Shift</kbd> + <kbd>Space</kbd></span> - Select current and previous items</li>
            </ul>
        </li>
        <li><b>Large Icons View</b>
            <ul>
                <li><span class="key-class"><kbd>Arrow Keys</kbd></span> - Navigate in directional movement</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Arrow Keys</kbd></span> - Move focus directionally</li>
                <li><span class="key-class"><kbd>Shift</kbd> + <kbd>Arrow Keys</kbd></span> - Select items directionally</li>
            </ul>
        </li>
        <li><b>Details View</b>
            <ul>
                <li><span class="key-class"><kbd>Up/Down Arrow</kbd></span> - Navigate files vertically</li>
                <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Up/Down Arrow</kbd></span> - Move focus vertically</li>
                <li><span class="key-class"><kbd>Shift</kbd> + <kbd>Up/Down Arrow</kbd></span> - Select files vertically</li>
            </ul>
        </li>
    </ul>
</ActionDescription>
