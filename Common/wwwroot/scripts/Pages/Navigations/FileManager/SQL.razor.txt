<ActionDescription>
<p>This sample demonstrates integration of the <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/sql-server-database-aspcore-file-provider" aria-label="GitHub sample link of SQL server file system provider">SQL server file system provider</a> with the File Manager component for database-backed file management.</p>
    
    <p>Supported file operations include create, rename, delete, refresh, and more. The SQL database provider manages file systems stored in SQL database tables using ID-based operations for unique file and folder identification.</p>
    
    <p>Setup requirements:</p>
    
    <ul>
        <li>Configure SQL server database connection via <code>SetSQLConnection</code> method</li>
        <li>Provide connection string in <code>Web.config</code> file</li>
        <li>Add connection entry in <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/sql-server-database-aspcore-file-provider/blob/master/appsettings.json" aria-label="GitHub sample link of JSON file">appsettings.json</a></li>
        <li>Specify connection name, table name, and table ID in <code>SetSQLConnection</code> method for file operations</li>
    </ul>
    
    <p>Database configuration uses <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/sql-server-database-aspcore-file-provider/tree/master/App_Data" aria-label="GitHub sample link of File Manager">FileManager.mdf</a> database file.</p>
    
    <p>Refer to the <a target="_blank" rel="noopener" href="https://github.com/SyncfusionExamples/sql-server-database-aspcore-file-provider">SQL server file system provider repository</a> for implementation details.</p>
</ActionDescription>
