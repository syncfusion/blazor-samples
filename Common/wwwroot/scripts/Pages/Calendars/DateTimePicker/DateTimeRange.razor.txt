<ActionDescription>
<p>
    This sample demonstrates date and time range selection in the DateTimePicker component. Date and time selection is restricted within specific ranges defined by the <code>Min</code>, <code>Max</code>, <code>MinTime</code>, and <code>MaxTime</code> properties. All other dates and times are out of range and restricted.
  </p>
  <p>
    The Date Range example explains date and time selection within specific ranges using the <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Calendars.DateTimePickerModel-1.html#Syncfusion_Blazor_Calendars_DateTimePickerModel_1_Min" target="_blank" rel="noopener" aria-label="Class reference of Min property in DateTimePicker">Min</a>, <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Calendars.DateTimePickerModel-1.html#Syncfusion_Blazor_Calendars_DateTimePickerModel_1_Max" target="_blank" rel="noopener" aria-label="Class reference of Max property in DateTimePicker">Max</a>, <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Calendars.DateTimePickerModel-1.html#Syncfusion_Blazor_Calendars_DateTimePickerModel_1_MinTime" target="_blank" rel="noopener" aria-label="Class reference of MinTime property in DateTimePicker">MinTime</a>, and <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Calendars.DateTimePickerModel-1.html#Syncfusion_Blazor_Calendars_DateTimePickerModel_1_MaxTime" target="_blank" rel="noopener" aria-label="Class reference of MaxTime property in DateTimePicker">MaxTime</a> properties.
  </p>
  <p>
    In this example, the first DateTimePicker range is restricted from 7th 10:00 AM to 27th 10:30 PM, and the second DateTimePicker time range is restricted from 10:00 AM to 8:30 PM daily.
  </p>
  <p>
    <strong>See also:</strong>
  </p>

  </ul>
</ActionDescription>

                <label class="example-label">DateTime Restriction</label>
                <SfDateTimePicker TValue="DateTime?" Value="@DateTimeValue" Min="@MinDate" Max="@MaxDate" ShowClearButton="true" FullScreen="true" Placeholder="Select a date and time" aria-label="Date and time selector with restrictions"></SfDateTimePicker>
            </div>
        </div>

            <div class="control-wrapper">
                <label class="example-label">Time Restriction</label>
                <SfDateTimePicker TValue="DateTime?" Value="@DateTimeValue" MinTime="@MinTime" MaxTime="@MaxTime" ShowClearButton="true" FullScreen="true" Placeholder="Select a date and time" aria-label="Time selector with restrictions"></SfDateTimePicker>
            </div>
        </div>
    </div>
</div>

@code {
    public DateTime DateTimeValue { get; set; } = new DateTime(DateTime.Now.Year, DateTime.Now.Month, 9, 11, 0, 0);
    public DateTime MinDate { get; set; } = new DateTime(DateTime.Now.Year, DateTime.Now.Month, 7, 10, 0, 0);
    public DateTime MaxDate { get; set; } = new DateTime(DateTime.Now.Year, DateTime.Now.Month, 27, 22, 30, 0);
    public DateTime MinTime { get; set; } = new DateTime(DateTime.Now.Year, DateTime.Now.Month, 7, 10, 0, 0);
    public DateTime MaxTime { get; set; } = new DateTime(DateTime.Now.Year, DateTime.Now.Month, 27, 20, 30, 0);
}
