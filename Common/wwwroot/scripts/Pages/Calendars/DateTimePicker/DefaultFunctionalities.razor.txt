<p>
    This sample demonstrates the default functionalities of the <a href="https://www.syncfusion.com/blazor-components/blazor-datetime-picker" aria-label="Feature Tour of Blazor DateTimePicker">DateTimePicker</a> component. Today's date is always highlighted in the popup calendar and receives focus if no date is selected. Select the desired date and time from the calendar and timepicker popup, and the selected date and time will be displayed.
  </p>
  <p>
    The DateTimePicker is a graphical user interface component that allows users to select or enter a datetime value through an interactive calendar and time picker popup.
