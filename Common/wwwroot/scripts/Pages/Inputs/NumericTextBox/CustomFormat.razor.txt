<ActionDescription>
<p>This example demonstrates the custom format functionalities of the Numeric TextBox component. The value of the Numeric TextBox will be displayed with the defined custom format while typing a value or changing a value using up/down buttons in the input boxes.</p>
    
    <p>The Numeric TextBox provides an option to customize the display format of the numeric value using the <a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Inputs.NumericTextBoxModel-1.html#Syncfusion_Blazor_Inputs_NumericTextBoxModel_1_Format" target="_blank" rel="noopener" aria-label="Class reference of Format property in NumericTextBox">Format</a> property. It accepts standard numeric format strings and custom numeric format strings as specified in <a target="_blank" rel="noopener" href="https://docs.microsoft.com/en-us/dotnet/standard/base-types/standard-numeric-format-strings" aria-label="Standard Numeric Format Strings documentation">MSDN</a>. The formatted value displays when the component is not focused.</p>
    
    <p>In this demo, the Numeric TextBox component is rendered with the custom format <b>###.## km</b>.</p>
    
    <p><b>See also</b></p>
    <ul>
        <li><a href="https://blazor.syncfusion.com/documentation/numeric-textbox/formats" target="_blank" rel="noopener" aria-label="Blazor NumericTextBox Number Formats documentation">Blazor Numeric TextBox - Number Formats</a></li>
    </ul>
</ActionDescription>


<div class="col-lg-12 control-section">
    <div class="control-wrapper">
        <div class="example-content">
            <label class="example-label">Distance</label>
            <SfNumericTextBox TValue="double?" Placeholder="Enter the distance" Value=250 Format="###.### km" Min=0 aria-label="Enter distance in kilometers"></SfNumericTextBox>
        </div>
        <div class="example-content">
            <label class="example-label">Tax</label>
            <SfNumericTextBox TValue="double?" Placeholder="Enter the tax" Format="### '%'" Value=25 Max=100 Min=0 aria-label="Enter tax in percentage"></SfNumericTextBox>
        </div>
        <div class="example-content">
            <label class="example-label">Price</label>
            <SfNumericTextBox TValue="double?" Placeholder="Enter the price" Format="$ ###.##" Value=1025 Min=0 aria-label="Enter the price in dollar"></SfNumericTextBox>
        </div>
    </div>
</div>
