<ActionDescription>
<p>This example demonstrates the default functionalities of the <a href="https://www.syncfusion.com/blazor-components/blazor-input-mask/" aria-label="Feature Tour of Blazor MaskedTextBox">MaskedTextBox</a> component. Enter a numeric value for the "Mobile number" textbox, then press tab key and enter a "Country ISO code" in alphabetic format. Next, type the "Date of birth" in numeric format, and finally enter the "Product key" in alphanumeric format.</p>
    
    <p>The MaskedTextBox is a textbox extended component that allows users to enter only valid input based on the provided mask. It is used to obtain input with specific formats such as phone number, date, time, IP, social security number, and more.</p>
    
    <p>MaskedTextBox supports a list of mask elements based on the standard <a target="_blank" rel="noopener" href="https://docs.microsoft.com/en-us/dotnet/api/system.windows.forms.maskedtextbox.mask?redirectedfrom=MSDN&view=net-5.0#System_Windows_Forms_MaskedTextBox_Mask">MSDN</a> mask elements.</p>

</ActionDescription>
