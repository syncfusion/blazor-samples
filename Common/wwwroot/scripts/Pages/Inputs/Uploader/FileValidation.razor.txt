<ActionDescription>
<p>This example demonstrates how to validate files before uploading them to the server. Only document files (DOC, DOCX, XLS, XLSX) with a minimum size of "10 KB" and maximum size of "100 MB" are allowed to upload to the server.</p>
    
    <p>The File Upload component allows you to validate the file's type and limit the file size using the following properties:</p>
    <ul>
        <li><a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Inputs.UploaderModel.html#Syncfusion_Blazor_Inputs_UploaderModel_AllowedExtensions" target="_blank" rel="noopener" aria-label="Class reference of AllowedExtensions property in File Upload">AllowedExtensions</a> - Specifies the file types allowed for upload.</li>
        <li><a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Inputs.UploaderModel.html#Syncfusion_Blazor_Inputs_UploaderModel_MinFileSize" target="_blank" rel="noopener" aria-label="Class reference of MinFileSize property in File Upload">MinFileSize</a> - Sets the minimum file size allowed.</li>
        <li><a href="https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Inputs.UploaderModel.html#Syncfusion_Blazor_Inputs_UploaderModel_MaxFileSize" target="_blank" rel="noopener" aria-label="Class reference of MaxFileSize property in File Upload">MaxFileSize</a> - Sets the maximum file size allowed.</li>
    </ul>
    
    <p><b>See also</b></p>
    <ul>
        <li><a href="https://blazor.syncfusion.com/documentation/file-upload/validation/" target="_blank" rel="noopener" aria-label="Blazor File Upload Validation documentation">Blazor File Upload - Validation</a></li>
    </ul>
</ActionDescription>


<div class="col-lg-12 control-section">
    <div class="control-wrapper">
        <SfUploader AllowedExtensions=".doc, .docx, .xls, .xlsx" MinFileSize=10000 MaxFileSize=104857600>
            <UploaderEvents OnRemove="OnFileRemove"></UploaderEvents>
            <UploaderAsyncSettings SaveUrl="https://blazor.syncfusion.com/services/production/api/FileUploader/Save" RemoveUrl="https://blazor.syncfusion.com/services/production/api/FileUploader/Remove"></UploaderAsyncSettings>
        </SfUploader>
    </div>
</div>
